# Claims
> Your go-to all-in-one land management plugin.

:::warning

Documentation is in development, more information will be added soon.

:::

### Server Requirements
**Server Software:** Paper or paper forks. \
**Server Version:** Latest only.

### Installation Process
1. Download the latest version [here](https://github.com/bradenk04/claims/releases)
2. Place the downloaded jar in the `plugins/` folder
3. Fully restart your server. *(reloads will not work, the server needs a full restart.)*
